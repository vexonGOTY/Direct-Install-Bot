    How it works

    1. 1 Fill in repository + token and save to localStorage.
    2. 2 Upload an IPA (to Filebin) or provide a direct URL.
    3. 3 Dispatch the main.yml workflow which commits the manifest to the generated branch.
    4. 4 Use the generated itms-services:// links to install on device.
