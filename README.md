
# Direct-Install-Bot
Automates direct install links
###### for private use only (if you want to use for yourself clone the repo)

first go to workflows and click on automate and install


click run and fill in the boxes - important, u need a **SIGNED** direct link



after filling in everything ( you can replace your previous one or if you want to have multiple, type in your folder name )

**IMPORTANT** do not use a folder name with spaces, use "_" or "-" instead.

last, copy the given link and paste it into safari.
